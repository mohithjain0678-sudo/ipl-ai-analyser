# 🏏 IPL AI Data Analyser

An AI-powered interactive cricket analytics dashboard built with Python, Streamlit, and LLaMA 3.3. Analyses 17 seasons of IPL data (2008–2024) with real-time charts and a natural language AI chatbot.

🔗 **Live Demo:** https://ipl-ai-analyser-ix6fjmujnxbjbk7hyowqfa.streamlit.app/

---

## Features

- 📊 Interactive charts — team win rates, venue analysis, season trends, player stats
- 🔽 Team selector dropdown — filter all stats by any IPL team instantly
- 🤖 AI Chatbot — ask natural language questions about IPL data powered by LLaMA 3.3 via Groq API
- 📈 1,095 matches and 260,920 ball-by-ball deliveries across 17 seasons
- ☁️ Deployed live on Streamlit Cloud
